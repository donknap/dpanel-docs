## 镜像地址

:::tip
根据实际的版本需求及网络环境，选择镜像地址替换示例中的镜像地址
:::

:::code-group

```shell [标准版镜像]
dpanel/dpanel:latest
```

```shell [Lite版镜像]
dpanel/dpanel:lite
```

```shell [阿里云标准版镜像]
registry.cn-hangzhou.aliyuncs.com/dpanel/dpanel:latest
```

```shell [阿里云Lite版镜像]
registry.cn-hangzhou.aliyuncs.com/dpanel/dpanel:lite
```
:::
